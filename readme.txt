== Description ==

WordPress Weather Widget.This plugin will show current day and next 4 days weather forecast.For more information,check out our <a href="http://show.yasiradnan.com">blog!</a><br />

<strong>Features:</strong>
<ul>
 <li>Easy to setup.You don’t need any kind of Programming.After activation,check the widget section and enter the name of the city.</li>
 <li>Well documented source code,you can easily customize the code</li>
 <li>Easy location input</li>
 <li>4 days weather Forecast.</li>
 <li>Our plugin included with custom icon.You can easily change the icon my replacing default icons on image folder.</li>
</ul>




== Installation ==

1. Upload the plugin folder to the "/wp-content/plugins/" directory of your WordPress site,
2. Activate the plugin through the 'Plugins' menu in WordPress,
3. Visit the "Widget" on administration page to setup the widget on sidebar.
